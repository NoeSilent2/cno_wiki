## Contributing

If you would like to contribute, you'll need to:
- Make a GitHub Account
- Create a fork of this repository
- Get GitHub Desktop
- Clone the fork you created on to your computer
- Create a new branch to work on
- Make whatever changes to the code you want
- Create a pull request from that branch
- Wait for me to decide to merge it

If you'd like to make sure I'll merge your PR (Pull Request), talk to me (NoeSilent) about what you're making.

Ideally, but this is not required, try to keep your PR's *atomic*. This means that it should only include changes pertaining to it.
For example, if you were making a new page for abilities, you would not want to package that with an image gallery of builds.
Both of those ideas are great, but should be added in separate PR's.

Mostly though, I'd like to work on optimizing the website. It currently sends excessive edge requests, and I'd like to keep this website inexpensive to run.
